El\u00a0sism\u00f2graf<\/a>\u00a0\u00e9s nom\u00e9s un comen\u00e7ament (explicaci\u00f3<\/span><\/a>). Creiem que amb les dades que tenim, es poden construir serveis i visualitzaciones molt m\u00e9s maques i \u00fatils. \u00a0<\/span><\/span><\/p>\n

Aqu\u00ed incloem algunes idees sobre les possibilitats que tenim amb el sism\u00f2graf:<\/span><\/span><\/p>\n

    \n
  • Construir un servei d’alerta per anunciar, per exemple:\u00a0“20\u00a0canvis\u00a0al Dept. de Just\u00edcia”<\/li>\n
  • Construir un robot-periodista que publica els canvis diaris a\u00a0un blog \u00a0i\u00a0\/ o a twitter<\/li>\n
  • Construir una corba de freq\u00fc\u00e8ncia\u00a0de canvis, semblant\u00a0a un sism\u00f2graf real<\/li>\n
  • Cercar les \u00e0rees m\u00e9s abandonades<\/li>\n
  • Cercar les persones amb el major nombre de\u00a0responsabilitats<\/li>\n<\/ul>\n

    Poseu-vos en contacte amb\u00a0@opengovcat<\/a>\u00a0si voleu construir alguna cosa. A sota ja podeu llegir la informaci\u00f3 b\u00e0sica de l’API.<\/p>\n

     <\/p>\n

    L’API del sism\u00f2graf<\/h2>\n

    De moment, l’API t\u00e9 nom\u00e9s un\u00a0endpoint:\u00a0<\/em><\/h4>\n
    http:\/\/18.184.0.9\/sismograf\/api\/canvis\/<departamento>\/<comienzo>\/<fin>\r\n<\/pre>\n
    Per exemple:<\/p>\n
    http:\/\/18.184.0.9\/sismograf\/api\/canvis\/all\/2014-01-01\/2014-01-31<\/a>\r\n<\/pre>\n
     <\/p>\n
    El resultat<\/h4>\n
    \u00a0{ status : \"ok\",\r\n   resultados : { borados    : [ (... lista de entidades borradas ...) ],\r\n                  canvisnom  : [ (... lista de entidades con nombres cambiados ...) ],          \r\n\r\n                  canvisresp : [ (... lista de entidades con responsabilidades cambiadas ...) ],\r\n\r\n                  nuevos     : [ (... lista de entidades nuevas ...) ]\r\n                 }\r\n }\r\n\r\n<\/pre>\n
    Propietats de les entitats:<\/p>\n
      \n
    • “centres”: Llista de centres per entidats amb diverses seus, valor copiat de la font.<\/li>\n
    • “cid”:\u00a0\u00a0ID intern del canvi. Amb la URL http:\/\/18.184.0.9\/sismograf\/c\/<id> es pot obtenir una representaci\u00f3 HTML.<\/li>\n
    • “class”:\u00a0\u00a0Classe CSS del\u00a0canvi<\/li>\n
    • “id”: “15377”:\u00a0ID de l’entitat. Amb la URL\u00a0http:\/\/www14.gencat.cat\/sacgencat\/AppJava\/organisme_fitxa.jsp?codi=<id> es pot\u00a0obtenir la fitxa de l’entitat<\/li>\n
    • “iddep”: “5823”: ID de l’entitat superior. Amb la URL\u00a0http:\/\/www14.gencat.cat\/sacgencat\/AppJava\/organisme_fitxa.jsp?codi=<id> es pot\u00a0obtenir la fitxa de l’entitat<\/li>\n
    • “nom”: Nom segons la font<\/li>\n
    • “oldnom” : Nom anterior<\/li>\n
    • “oldresp”: Responsable anterior<\/li>\n
    • “oldstamp”: Data quan es va\u00a0registrar per primera vegada el responsable o el nom anterior<\/li>\n
    • “resp”: Responsable actual<\/li>\n
    • “stamp”: Data\u00a0del canvi<\/li>\n
    • “test” : veure sota<\/li>\n<\/ul>\n
      Les entitats canviades tenen una propietat “test<\/strong>“, un nombre\u00a0entre 0 i 2; un valor major significa que hi va haver m\u00e9s canvis en el nom (canvisnom) o en el responsable (canvisresp). Alguns canvis\u00a0de responsables o noms s\u00f3n simplement correcions \u00a0– amb la propietat “test” \u00e9s possible diferenciar-ho. Els canvis amb “test”<0.1 s\u00f3n filtrats per l’API i no apareixen en el resultat. “test” es\u00a0calcula aix\u00ed:\u00a0Leveshtein edit distance(old,new) \/ Max(len(old),len(new)).<\/em><\/p>\n
